What is DevToolbox?

DevToolbox is a suite of everyday developer utilities that runs entirely in your browser. No npm install. No webpack. No login. No tracking. Just open a page, use the tool, close the tab.

Every tool is a single self-contained HTML file. The entire site weighs less than a typical JPEG. It works offline. It installs as a native app on your desktop or taskbar via PWA. And it never phones home.

Philosophy

⚡
Zero dependencies
No React, no Vue, no Tailwind, no lodash. Every tool is written in vanilla HTML, CSS, and JavaScript. The browser is the framework.
🔒
Privacy by architecture
There is no server to send data to. JWT tokens, Base64 strings, JSON — everything you paste stays in your browser's memory and is discarded when you close the tab.
📦
Offline-first
A service worker caches every page on first visit. After that, DevToolbox works with no internet connection — on a plane, in a basement, behind a corporate firewall.
⌨
Keyboard-first
Every tool is reachable in one keystroke. The command palette (⌘K) lets you jump anywhere without reaching for the mouse. Developers live in the keyboard.
🌙
Dark by default
Because your terminal is dark, your editor is dark, and every other tool that fights this is wrong.

Tech Stack

  • HTML / CSS / JavaScript — plain, standard, no transpilation
  • GitHub Pages — static hosting, zero infrastructure cost
  • Service Worker API — offline support and PWA installability
  • localStorage — persisting pinned tools, nothing else
  • Zero build tools — edit a file, push to git, done

Why I Built This

I got tired of Googling "base64 encode online" and landing on ad-bloated, login-walled, cookie-banner-filled sites that still managed to feel slow. Every one of those tools is trivial to implement. So I built them all in a weekend, made them fast, made them private, and made them installable.

The goal is a toolbox that feels like a native app — because it is one. You install it once from the browser and it's in your Dock, offline, forever, with no update prompts, no subscriptions, no dark patterns.
