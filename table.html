What is a Markdown Table?

Markdown tables use pipe characters (|) as column separators and a row of dashes (---) to separate the header from the data rows. They are rendered as visual tables on GitHub, GitLab, Notion, Obsidian, and most documentation platforms. 

| Name  | Age | City      |
| ----- | --- | --------- |
| Alice | 30  | Sydney    |
| Bob   | 25  | Melbourne |

When to Use Markdown Tables

  • GitHub README: Document API parameters, configuration options, or feature comparisons in a readable grid.
  • Pull request descriptions: Before/after comparisons of benchmark results or API responses.
  • Notion / Obsidian notes: Lightweight databases without needing a spreadsheet application.
  • Technical documentation: Reference tables for error codes, keyboard shortcuts, or CLI flags.

How to Use This Tool

Copy any range of cells from Excel or Google Sheets and paste directly into the input area. Spreadsheet applications copy cell data as tab-separated values (TSV), which this tool parses automatically. Toggle "Treat first row as header" to control whether the first row becomes a bold header row with the separator line beneath it.
